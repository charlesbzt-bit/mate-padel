# Mate (Padel)

Mate est une web app qui permet de créer une intention de match de padel et de trouver 3 autres joueurs avant de réserver.

## Features
- Auth (Supabase)
- Créer une intention (date, créneau, zone, niveau)
- Rejoindre / quitter une intention
- Compteur 4/4 + proposition de clubs

## Tech
- React + TypeScript + Vite
- Supabase (Auth + Postgres)
